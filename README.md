# .NET Conf Taiwan 2023 - 範例程式

## 課程名稱：Blazor in .NET 8 的重大改變

## 範例程式內容

商業應用開發：以 WebApp 一路打通 Full-Stack 全端開發 的 從無到有系列的『完整範例程式』

在這個 Live Demo 裡，我在全新的 Blazor 的 WebAssembly 的混和模式裡，展示自定義的 API Framework 與 Blazor 混和模式的結合，利用 Server Side 專案建立 API Controller 後，在展示一種使用 NuGet Packages包裝 Blazor Components 的情境，透過安裝即可使用。

另外，範例中的 GridView 的 Blazor Component 我故意包裝成 .NET 6 版本的，這展示另一種企業內常見的情境，在 .NET 8 相容使用舊有的 .NET 6 的 Blazor Components 的情況，以呼應我在倒數第 4 張投影片裡的【Blazor 元件開發者的注意事項】

下面是這個課程的所有內容。

## 課程的全部章節

* 初探 Blazor in .NET 8
* 全新範本 Blazor United Project (Web App)
  - 統一所有 Blazor 開發的全新樣板
  - 有那些東西不見了？
* Server-side Rendering 與 Streaming Rendering
  - WebApp 如何取代先前我提的 Hosted by Server？
* 如何將 Blazor in .NET 6/7 移轉至 .NET 8
  - 案例介紹：將 .NET 6 的 BlazorApp 無痛移轉到 .NET 8
* Blazor 元件開發者的注意事項
* 商業應用開發：以 WebApp 一路打通 Full-Stack 全端開發
